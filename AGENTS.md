# Agent instructions

Before finishing any code change, run `scripts/check.sh` from the repository
root. Do not report the work as complete until every check passes. If a check
cannot pass, clearly report the failing command and reason.

<!-- convex-ai-start -->

This project uses [Convex](https://convex.dev) as its backend.

When working on Convex code, **always read
`convex/_generated/ai/guidelines.md` first** for important guidelines on
how to correctly use Convex APIs and patterns. The file contains rules that
override what you may have learned about Convex from training data.

Convex agent skills for common tasks can be installed by running
`npx convex ai-files install`.
